# Obsidian LLM Wiki

Этот репозиторий хранит Obsidian vault для ведения локальной LLM-ориентированной базы знаний.

Начинайте с `wiki/index.md`, затем переходите по связанным тематическим страницам. Исходные или слабо обработанные материалы находятся в `raw/`.

## Структура

- `wiki/` - обработанные заметки базы знаний и индексные страницы.
- `raw/` - исходные или слабо обработанные заметки-источники.
- `prompts/` - рабочие промпты для LLM-агента, который поддерживает vault.
- `.obsidian/` - общие настройки Obsidian для этого vault.
- `.gitignore` - исключает локальное состояние рабочего пространства, кэши, системные и временные файлы.

## Что отслеживать в Git

Коммитьте заметки, исходные материалы, вложения и осознанно выбранные общие настройки Obsidian.

Не коммитьте локальное состояние рабочего пространства, кэши, временные файлы и настройки плагинов, если они содержат API-ключи, токены, учетные данные или приватные данные сервисов.

Перед коммитом настроек community-плагинов проверяйте файлы вроде `.obsidian/plugins/*/data.json` на наличие секретов.

## Работа c Vault

Откройте эту папку как Obsidian vault. Используйте `wiki/index.md` как основную точку входа для навигации, а `raw/` - когда нужно проверить или пересмотреть исходные материалы.

## Промпты

- `prompts/initial-llm-wiki-prompt.md` - базовый промпт для запуска агента как смотрителя LLM-Wiki.
- `prompts/raw-change-workflow-prompt.md` - промпт для обработки добавленных или удаленных файлов в `raw/`.
