# PARA

PARA - метод организации Second Brain от Tiago Forte: Projects, Areas, Resources, Archives.

## Компоненты

- Projects - активные краткосрочные цели с результатом.
- Areas - долгосрочные зоны ответственности, например здоровье или карьера.
- Resources - справочные материалы и идеи.
- Archives - завершенные или неактивные материалы.

## Где полезен

PARA помогает быстро решить, куда положить повседневную заметку. Для личного [[Second Brain]] это снижает трение и предотвращает бесконечное проектирование структуры.

## Ограничение для LLM Wiki

Для [[LLM Wiki]] PARA не заменяет `raw/` и `wiki/`. Источники должны оставаться неизменяемыми в `raw/`, а обработанное знание - жить в `wiki/`. PARA можно использовать как дополнительную навигацию для личных проектов, но не как основной ingest-протокол.
